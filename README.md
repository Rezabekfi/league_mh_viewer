# league_mh_viewer

App for viewing your match history from all of your league of legends accounts at once. Purpose of this app is to make it easier for me to track my progress on mine accounts but also to have a working match history tab (which often bugs out in the official client). 

## Secondary goals

- To learn how to use the Riot API (and to learn how to use APIs in general) *note: I have prio experiences with APIs, but I want to have it publicly available for potential employers to see*
- To learn how to use C# properly (I have very minor experiences with C#)
